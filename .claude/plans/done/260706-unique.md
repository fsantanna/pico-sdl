# unique : layer constructors return key + pico_id

## Goal

- All layer constructors return their created key.
- `pico_id` returns a unique monotonic integer.
- `key` becomes optional in all constructors: when absent, a
  unique key `pico-N` is auto-generated via `pico_id`.

## Design

| Item        | Decision                                             |
|-------------|------------------------------------------------------|
| `pico_id`   | `int pico_id (void)` — counter from 1, no alloc      |
| `pico.id`   | `pico.id()` → integer                                 |
| Return      | `pico_layer_*` (+ `_mode`) return `const char*`      |
| Storage     | returned pointer is the realm's internal key copy    |
| Auto key    | `key==NULL` → `snprintf("/id-%d", pico_id())`        |
| Namespace   | `/` prefix is user-reserved → no collision possible  |
| image/video | path default wins: `key==NULL` → path (existing)     |
| Uniqueness  | counter only; never reset in `pico_init`             |
| Lua         | all 7 `pico.layer.*` push key, `return 1`            |
